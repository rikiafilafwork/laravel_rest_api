# Deskripsi
Sample Rest API Menggunakan Laravel

# Clone App
```
git clone https://github.com/rikiafilafwork/laravel_rest_api.git
```

# Composer Install
```
composer install
```

## Buat .env file
```
cp .env.example .env
```
# Generate Key
```
php artisan key:generate
```
# Migration & Seeder
```
php artisan migrate --seed
```
# Run Test Unit
```
php artisan test
```
